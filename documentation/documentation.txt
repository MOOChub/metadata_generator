<h2>How to</h2>
  <p>This tool was developed to support course creators and admins to generate metadata for their courses. It generates
  data fragments according to the MOOChub metadata format.</p>

  <p>The user can choose a framework and selects the desired field of study of skill via the dropdown menus.</p>

  <p>By clicking the "Add Entry" button will add the selected field of study or skill to the list which will be used for
  the generation of the metadata fragment(s).</p>

  <p>By clicking the "Write JSON" button the metadata fragment(s) will be generated and can be downloaded. Important:
  fields of study and skills are saved in separate JSON files.</p>

  <p>Single entries from the list can be deleted by pressing the "Delete Entry" button. The list can be completely
  cleared by pressing the "Reset" button.</p>
